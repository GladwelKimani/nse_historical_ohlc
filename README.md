# NSE Historical OHLC Data

## Description
This repository contains historical OHLC (Open, High, Low, Close) trading data for 52 companies listed on the Nairobi Securities Exchange (NSE).

## Data Structure
- **Format:** CSV
- **Columns:** Date, Open, High, Low, Close, Volume
- **Range:**  01/01/2015  to 11/28/2025
